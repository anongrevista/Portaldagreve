README_FOLDER.txt - Pasta src/

Este é o coração do código fonte da aplicação.

- app/: Contém as rotas, layouts e páginas da aplicação utilizando o Next.js App Router.
- components/: Biblioteca de componentes React (UI) modulares e reutilizáveis.
- lib/: Configurações de bibliotecas externas e funções utilitárias core.
- store/: Gerenciamento de estado global da aplicação (Zustand).
- utils/: Funções de ajuda e lógica de integração com serviços externos como o Supabase.
