README_FOLDER.txt - Pasta src/app/(dashboard)/

Esta pasta agrupa as páginas que compõem o Dashboard principal da Central.

- calendario/: Página com a visão completa do cronograma de atividades.
- documentos/: Rota dinâmica que renderiza os textos e ofícios a partir de slugs.
- submit/: Página para envio e submissão de novos documentos para a Central.
- layout.tsx: Define a estrutura visual persistente do dashboard (Sidebar + TopBar + Conteúdo).
- page.tsx: A página inicial (Home) do Dashboard.
