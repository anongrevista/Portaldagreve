README_FOLDER.txt - Arquivos na Raiz do Projeto

Nesta pasta estão os arquivos de configuração fundamental do ambiente de desenvolvimento e build da aplicação.

- next.config.mjs: Arquivo de configuração do Next.js para definição de roteamento, imagens e otimizações.
- package.json: Define as dependências do Node.js, scripts de execução (dev, build, start) e metadados do projeto.
- package-lock.json: Registro exato das versões de todas as dependências instaladas.
- postcss.config.mjs: Configuração do PostCSS, responsável por processar o Tailwind CSS.
- tailwind.config.ts: Configuração mestre do Tailwind CSS, incluindo definição de cores da marca, fontes e extensões de tema.
- tsconfig.json: Configurações do compilador TypeScript para garantir o rigor técnico do código.
- next-env.d.ts: Declarações de tipos automáticas do Next.js.
